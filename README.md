## 📌 Overview

**Streamlit Bank Management System** is a banking application built using **Python** and **Streamlit**. It provides a simple and interactive interface for managing bank accounts and performing common banking operations.

---

## ✨ Features

- 🆕 Create New Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 👤 View Account Details
- ✏️ Update Customer Information
- ❌ Delete Account
- 🔐 Secure 4-Digit PIN
- 💾 JSON-Based Data Storage
- 🌙 Clean Dark Theme UI

---

## 🛠️ Technologies Used

- 🐍 Python 3
- 🎈 Streamlit
- 📄 JSON
- 💻 VS Code

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/ritesh-chaudhary-ai/streamlit-bank-system.git
```

Move into the project folder:

```bash
cd streamlit-bank-system
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
